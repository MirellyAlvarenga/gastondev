## Rodando localmente

* Clone o repositório e instale as dependências.
```bash
  git clone https://github.com/GastonZinhow/gastondev.git
  cd gastondev
  npm install
```
* Inicie o servidor de desenvolvimento
```bash
  npm run dev
```
